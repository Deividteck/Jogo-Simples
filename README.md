# 🎯 Jogo de Adivinhação (1 a 50)

Um jogo simples de tentativa e erro onde o jogador precisa adivinhar um número aleatório entre **1 e 50**, tendo apenas **5 chances**.

---

## 🎮 Como funciona

* O sistema gera um número aleatório entre **1 e 50**
* O jogador tenta adivinhar o número
* A cada tentativa, o jogo informa:

  * 🔼 Se o número é maior
  * 🔽 Se o número é menor
* O jogador tem **5 tentativas no total**
* O jogo termina quando:

  * ✅ O jogador acerta o número
  * ❌ Ou as tentativas acabam

---

## 🚀 Funcionalidades

* 🎲 Geração de número aleatório
* 🔢 Entrada de tentativa do usuário
* 💬 Feedback em tempo real (maior/menor)
* ⏳ Limite de tentativas
* 🏁 Mensagem de vitória ou derrota
